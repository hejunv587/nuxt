# 知识点
## git
* 将本地分支 local-branch 与远程分支 remote-branch 关联起来：<br/>  git branch --set-upstream-to=origin/remote-branch local-branch
* 拉取和推送到远程分支 <br/>
 git pull origin remote-branch:local-branch<br/>
 git push origin local-branch:remote-branch
